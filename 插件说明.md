## 安装说明

### 方法一：通过feeds安装（推荐）

```bash
# 添加源到feeds.conf.default
sed -i '$a src-git jell https://github.com/wwz09/QCA-Package' feeds.conf.default

# 更新feeds
./scripts/feeds update -a

# 安装所有插件
./scripts/feeds install -a

# 或者安装特定插件
./scripts/feeds install luci-app-xxx
```

### 方法二：手动编译

1. 将插件目录复制到OpenWrt SDK的package目录
2. 运行 `make menuconfig` 选择需要的插件
3. 运行 `make package/xxx/compile V=s` 编译插件

### 方法三：下载预编译包

访问项目的Releases页面下载预编译的ipk包，然后在路由器上安装：

```bash
opkg install xxx.ipk
```

2. **依赖安装**：某些插件可能需要先安装依赖包

4. **资源占用**：注意插件的资源占用，避免影响路由器性能
